# Stamp-a-Day 설치 가이드 📮

매일 사진 한 장을 우표로 만들어 월별 그리드에 붙이는 일기 앱이야.
아직 개발 중이라 앱스토어에는 없고, 아래 방법으로 네 컴퓨터 + 아이폰으로 실행해볼 수 있어.

## 준비물

- **컴퓨터** (윈도우 또는 맥) — 개발 서버를 띄우는 용도
- **아이폰** — 앱이 실제로 뜨는 곳
- 컴퓨터와 아이폰이 **같은 Wi-Fi**에 연결되어 있어야 함 (중요!)

## 1. 컴퓨터에 설치할 것

### Node.js
1. https://nodejs.org 접속
2. **LTS** 버전 다운로드해서 설치 (전부 기본값으로 다음다음 누르면 됨)
3. 설치 확인: 터미널(윈도우는 PowerShell, 맥은 터미널)을 열고
   ```
   node --version
   ```
   입력했을 때 `v22.x.x` 같은 버전이 나오면 성공

### Git
- 윈도우: https://git-scm.com/download/win 에서 설치
- 맥: 터미널에 `git --version` 쳐보고 없다고 하면 뜨는 안내대로 설치

## 2. 아이폰에 설치할 것

- App Store에서 **Expo Go** 검색해서 설치 (무료)
- 별도 회원가입 필요 없음

## 3. 프로젝트 받기 & 실행

터미널을 열고 아래를 한 줄씩 입력해:

```bash
# 1) 프로젝트 다운로드
git clone https://github.com/nhjin/stampProject.git

# 2) 폴더로 이동
cd stampProject

# 3) 필요한 패키지 설치 (1~2분 걸림, 최초 1회만)
npm install

# 4) 개발 서버 실행
npm start
```

잠시 기다리면 터미널에 **큰 QR 코드**가 떠.

## 4. 아이폰으로 열기

1. 아이폰 **기본 카메라 앱**을 켜고 터미널의 QR 코드를 비춰
2. 화면에 뜨는 "Expo Go에서 열기" 배너를 탭
3. 잠깐 로딩 후 하늘색 배경의 우표 그리드가 뜨면 성공! 🎉

### 써보는 법
- **오늘 날짜 칸**(주황 점선 칸)을 탭 → 사진 선택 → 한 줄 메모 → **붙이기**
- 지나간 날짜의 빈 칸도 채울 수 있어
- 붙인 우표를 탭하면 크게 볼 수 있어

## 안 될 때 (문제 해결)

| 증상 | 해결 |
|------|------|
| QR 스캔해도 반응 없음 | 컴퓨터와 아이폰이 같은 Wi-Fi인지 확인. 그래도 안 되면 `npm start` 끄고(`Ctrl+C`) `npx expo start --tunnel` 로 다시 실행 (인터넷 경유라 좀 느리지만 Wi-Fi 달라도 됨) |
| "최신 버전 Expo Go 설치" 문구 | App Store에서 Expo Go 업데이트 후 다시 시도 |
| 이상한 에러로 로딩 실패 | 서버 끄고 `npx expo start -c` 로 캐시 지우고 재시작 |
| 윈도우 방화벽 팝업이 뜸 | "액세스 허용" 눌러주기 |

## 끄는 법

- 터미널에서 `Ctrl+C` 누르면 서버 종료
- 다음에 또 볼 땐 `cd stampProject` → `npm start` 만 하면 됨 (설치 과정 생략)
